# Sales Analysis Project

## Overview
This project analyzes sales data using Python and Jupyter Notebook.  
It includes:
- Region-wise sales
- Category-wise profit
- Top-selling products
- Charts & visualizations

## Dataset
The dataset used: `sales_raw.xlsx` (500 rows, 10 columns)

## Steps
1. Load dataset using pandas
2. Check data info and statistics
3. Perform analysis:
   - Total Sales per Region
   - Total Profit per Category
   - Top Selling Products
4. Visualize results using Matplotlib

## Tools
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook

## Output
All analysis results and charts are included in `analysis.ipynb`.
